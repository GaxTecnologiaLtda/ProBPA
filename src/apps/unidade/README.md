# Painel da Unidade (Unit Panel)

This module handles the workflow for Basic Health Units (UBS/ESF), focusing on:
- Reception & Scheduling
- Attendance Queue Management
- Professional Scales/Rosters
- Unit Management Dashboard

## Structure
- `components/`: Local components
- `pages/`: Main views (Dashboard, Schedule, List, Roster)
- `hooks/`: Unit-specific logic
- `context/`: State management for the unit session

## Reference
Based on e-sus APS PEC guidelines, adapted for ProBPA identity.
